### Penjelasan Program:
Program ini dibuat untuk memenuhi tugas Posttest PBO. Program berfungsi untuk mencatat data senjata pada zaman medieval.

### Fitur:
1. **Tambah Data:** Menambahkan objek senjata (nama, tipe, damage) ke dalam ArrayList.
2. **Lihat Data:** Menampilkan semua list senjata yang sudah diinputkan.
3. **Edit Data:** Mengubah data senjata yang sudah ada berdasarkan nomor index-nya.
4. **Hapus Data:** Menghapus data senjata dari list.
5. **Looping:** Program menggunakan `while(true)` agar bisa terus berjalan sampai user memilih menu keluar.

### Cara Menjalankan:
1. Compile file `Senjata.java` dan `Main.java`.
2. Jalankan `Main.java`.
3. Masukkan pilihan menu 1-5.
